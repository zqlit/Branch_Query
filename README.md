## 项目简介

该项目是一个模拟的中国邮政集团有限公司网站，旨在展示 Vue.js 的基本使用和一些常见的前端功能实现。

## 技术栈

- Vue.js
- Vue Router
- Axios
- LESS
- Normalize.css

## 安装步骤

1. 克隆项目到本地：

    ```bash
    git clone https://github.com/yourusername/your-repo-name.git
    ```

2. 进入项目目录：

    ```bash
    cd your-repo-name
    ```

3. 安装依赖：

    ```bash
    npm install
    ```

    或者使用 Yarn：

    ```bash
    yarn install
    ```

## 使用方法

1. 启动开发服务器：

    ```bash
    npm run serve
    ```

    或者使用 Yarn：

    ```bash
    yarn serve
    ```

2. 打开浏览器并访问 `http://localhost:8080`。
